Создание безусловных рефлексов без коннекта с Beast

 

Редактор позволяет системно и быстро создавать новые безусловные рефлексы.

Системно потому, что при выборе основных условий (в виде Базового состояния и сочетания Базовых контекстов) создается таблица со всеми возможными состояниями пусковых стимулов (действий с Пульта). В таблице используются до 2-х всех допустимых сочетаний пусковых стимулов (ПС), исключая антагонистов (несовместимых) ПС.

Остается дополнить правую колонку перечислением действий.

Если для данного сочетания условий уже есть рефлекс, то он отмечается вставкой ID в первой колонке и его список действий невозможно исправить здесь (можно только в основном редакторе рефлексов).

Использование:

В верхнем выпадающем списке выбрать Базовый контекст и под ним – выбрать одно из сочетаний Базовых контекстов, которые станут условиями рефлекса.

После нажатия кнопки “Создать таблицу для заполнения рефлексами” будет сформирована таблица, в правой колонке которой нужно выбрать список действий. После заполнения таблицы следует нажать под ней “Сохранить рефлексы”, после чего новые рефлексы будут дописаны после старых.

Пояснение:

В редакторе Гомеостаза есть таблица “Активности Базовых стилей”, в каждой строке которой определяется, какие из Базовых контекстов (стилей поведения) будут активны при определенных состояниях данного параметра гомеостаза.

Каждый параметр гомеостаза имеет зону выхода параметра из нормы (определяется в таблице “Жизненные параметры гомеостаза”), а оставшийся участок нормы поделен на 5 равных диапазонов нормы.

Еще одно состояние “Возврат в норму” у параметра возникает, если ранее у него было состояние “Выход из нормы”, но значение параметра улучшилось. Это принципиально не позволяет определить возможные сочетания Базовых контекстов для Базовых состояний (Хорошо, Норма и Плохо). Поэтому в алгоритме выборки всех возможных сочетаний контекстов Базовое состояние (самый верхний выпадающий список) не может принимать участия.

При разных сочетаниях разных состояний параметров гомеостаза (затрагивающих разные строки таблицы) получается набор тех или иных Базовых контекстов. Эти наборы упорядочиваются: убираются повторения, антагонисты и убираются гасящиеся контексты (те, что в таблице прописаны со знаком минус), а результатом заполняется второй выпадающий список условий.

Второй список условий – заведомо избыточный и поэтому для каждого из Базовых состояний верхнего списка нужно разумно выбирать условие второго списка для заполнения формирующейся таблицы возможных рефлексов.

Так же таблица возможных рефлексов – избыточна и не стоит для всех строк заполнять действия, а только для тех, которые разум подсказывает реально возможными или желаемыми для рефлекторного поведения Beast.
