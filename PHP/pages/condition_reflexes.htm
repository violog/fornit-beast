Формирование набора условных рефлексов.

Период взаимодействия с Beast любым образом, с разными сочетаниями действий и очень простых фраз в различных состояниях его Базовых параметров (для этой стадии развития можно устанавливать слайдерами Пульта). Чем дольше этот период, тем более эффективные навыки получит Beast.

Чтобы создался новый рефлекс нужно не менее 3-х раз повторить воздействие пусковых стимулов, не обязательно подряд, - этим предотвращаются случайные, мусорные сочетания. Для формирования условных рефлексов нужно потратить немало времени (в случае ребенка около года). Но если на Пульте над полем ввода фраз для Beast проставить галку “набивка рабочих фраз без отсеивания мусорных слов”, то у словные рефлексы будут образовываться за один раз.

Быстрое создание базы условных рефлексов

Есть способ быстрого создания базового набора условных рефлексов. Для этого нужно воспользоваться редактором фраз-синонимов условных рефлексов http://go/pages/condition_reflexes_basic_phrases.php, у которого есть свой топик пояснений.

После набивки, в редакторе условных рефлексов http://go/pages/condition_reflexes.php нужно нажать кнопку “Сформировать условные рефлексы” и дождаться конца процесса.

На пульте будет отслеживаться списки слов для каждого сочетания текущих условий и если они не пусты, то справа от “Послать сообщение Beast” будет появляться иконка, при щелчке по которой можно выбрать слова-синонимы условных рефлексов, и они будут вставляться в поле ввода фраз.
